JavaScript dasturlsh tilida ozgaruvchilarni 3 xil usulda elon qilish imkoniyati mavjud: let,const va var kalit sozlaridan foydalanib oshiriladi.

LET

let-dasturimiz ishlashi davomida oagarishi mumkin bolgan ozgaruvchilarni elon qilish uchun foydalaniladi.

❗️ Elon qilinib bolingan ozgaruvchini qiymatini ozgartirishda hech qanday kalit sozlardan foydalanilmaydi.

CONST

const-dasturimiz ishlashi davomida ozgarmaydigan ozgaruvchilarni elon qilish uchun foydalaniladi.

❗️ const yordamida elon qilingan ozgaruvchilar har doim boshlang'ich qiymatga ega bolishi shart.

VAR

var * - bu eski turdagi o'zgaruvchi bo'lib, ikkita bir xil o'zgaruvchi e'lon qilish mumkin.
var a = 1
var a = 2 letkin const hamda let da bu mumkin emas.

var ni o'zidan oldin ishlatish mumkin va javob - undefined
console.log(a) // undefined
var a = 1
